Basic Toggle Action

```js
<ToggleAction>
  Fullscreen
</ToggleAction>
```

Toggled

```js
<ToggleAction isToggled>
  Fullscreen
</ToggleAction>
```

Toggle Action with icon
```js
<ToggleAction activeIcon="compress" inactiveIcon="expand">
  Fullscreen
</ToggleAction>
```

Toggled with icon
```js
<ToggleAction isToggled activeIcon="compress" inactiveIcon="expand">
  Fullscreen
</ToggleAction>
```
